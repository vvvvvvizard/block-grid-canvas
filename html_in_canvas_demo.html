HTML-in-Canvas API

Bringing full DOM interactivity, accessibility, and standard browser features inside a high-performance `` element pixel grid.

Prerequisites to Try This Feature:

  1. Use Google Chrome Canary (v149 or later).
  2. Type chrome://flags/#canvas-draw-element in your address bar.
  3. Change the flag to Enabled and click Relaunch.
  4. If the flag is not enabled, the canvas will fall back to showing standard static HTML, but the interactive rendering code won't run.
Note: Inspect this page's source code to see how the layoutsubtree attribute, ResizeObserver, and drawElementImage are implemented.
